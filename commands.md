## Most common ones

Remove a file
~~~bash
$ rm 'filename'
$ rm ./path/to/the/file
~~~

<br>

Move a file
~~~bash
$ mv 'filename_or_directory' /destination/root
~~~

<br>

Rename a file (same for directories)
~~~bash
$ mv 'filename' 'newname'
~~~

<br>

Show elements on this directory

```bash
$ ls
```

Create files
~~~bash
$ touch 'filename'
~~~


---

Install something

~~~bash
$ sudo snap install 'program' --setting
~~~
